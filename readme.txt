== Overview ==

This repository contains the "theone" WordPress theme, a landing page theme created from an HTML/SASS template. It features a clean and modern design and is built to be fully responsive across all devices.

== Features ==

- Responsive Design: Ensures a seamless experience on all devices, from desktop to mobile.
- Modern Layout: Clean and contemporary design with a grid-based layout.
- Customizable: Easily adapt the theme to fit your branding and content needs.
- Smooth Transitions: Includes smooth transitions and animations for a polished look.

== Installation ==

To install and activate this theme:

1. Download the theme from the repository.
2. Upload the `theone` folder to the `/wp-content/themes/` directory of your WordPress installation.
3. Go to the WordPress admin panel, navigate to "Appearance" -> "Themes".
4. Find "theone" in the list of themes and click "Activate".

== Customization ==

To customize the theme:

1. Open the `style.css` file to modify styles.
2. Edit the `functions.php` file for theme functionality.
3. Adjust theme settings in the WordPress Customizer.
